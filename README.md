# VerdeLar

É um projeto interdisciplinar da faculdade (Fatec-Cruzeiro) desenvolvido no meu **2º semestre, em 2025**, com foco em **ODS 2 – Combate à Fome**.

## Sobre o projeto:
O **VerdeLar** é um sistema digital educativo que permite:

- Visualizar informações sobre **plantio** e cuidados com diferentes plantas.  
- Adicionar plantas a um **jardim digital**, registrando nome e descrição.  
- Realizar **lições e desafios** para evoluir as plantas virtualmente.  
- Explorar diferentes tipos de **plantas, solos e espaços**.

> As plantas são adquiridas na vida real; o sistema apenas registra e acompanha seu crescimento digital.

## Tecnologias utilizadas:
- **Java**  
- **Maven** (`pom.xml`)  
- Banco de dados `.sql` apenas para **estrutura** (sem dados sensíveis)  

## Considerações finais:
O VerdeLar une **programação, educação ambiental e conscientização social**, incentivando o cuidado com plantas e a reflexão sobre segurança alimentar.
